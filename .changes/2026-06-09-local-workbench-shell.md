Added a dependency-free local static workbench shell under `apps/workbench` that renders
current SDK runtime evidence, explicit CLI state summaries when provided, local approval
records, artifacts, traces, memory/context, generated docs previews, and the generated
system map. Hosted workbench, hosted stores, Studio UI package integration, Mintlify
validation/build/publish, and release publishing remain fail-closed unsupported.
